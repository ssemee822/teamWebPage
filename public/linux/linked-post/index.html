Linux 기본 명령어

By 안승환 August 1, 2021

• ls : 현재 디렉토리의 파일과 폴더 목록을 표시합니다. ls -l : 파일에 대한 정보를 길게 출력 ls -al : 숨긴 파일을 자세하게 표시

• cd : 디렉토리 변경을 위해 사용합니다. cd .. : 상위 디렉토리로 이동 cd - : 이전에 있던 디렉토리로 이동 cd ~ : 홈 디렉토리로 이동 cd / : 루트 디렉토리로 이동 cd <경로> : <경로>로 이동

• pwd : 현재 작업중인 디렉토리의 경로를 표시합니다.

• mkdir : 새로운 디렉토리를 생성합니다

• rm : 파일이나 디렉토리를 삭제합니다.

• mv : 파일이나 디렉토리를 이동하거나 이름을 변경합니다.

• cat : 파일의 내용을 표시합니다.

• grep : 텍스트에서 패턴을 검색합니다.

• touch : 빈 파일을 만듭니다.

• chmod : 파일이나 디렉토리의 권한을 변경합니다.

• chown : 파일이나 디렉토리의 소유자를 변경합니다.
